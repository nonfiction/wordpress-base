# PHP/Apache base image for Wordpress

This extends `php:8.2-apache` with a few additional packages we rely on for
deploying Wordpress websites.

## Usage:

```dockerfile
FROM nonfiction/wordpress-base:v5
# ...
```

## Makefile commands:  

```
make update
make build
make push
make shell
```
